IPL Match Data Analysis 🏏
📌 Project Overview

This project performs data analysis on Indian Premier League (IPL) matches using Python. The program generates a synthetic IPL dataset and analyzes different aspects of matches such as team performance, toss impact, venue scoring patterns, and player achievements.

The project demonstrates basic Data Science techniques including data generation, data analysis, and data visualization.

🚀 Features:
Generate synthetic IPL match dataset
Team performance analysis
Toss impact analysis
Venue scoring analysis
Player of the Match analysis
Head-to-head team comparison
Data visualization using charts

Technologies Used:
Python
Pandas
NumPy
Matplotlib
Seaborn
